This MERN-TASK-MANAGER is a task managment web application, in which admin can assign task and their Due date to Developer. Developer can
See his own assigned task and mark them completed when done. Admin can create and remote the developer and Task assigne to the developer.
To start the application on local machine.

Open code in IDE 
Run command : npm run dev  "for frontend"

open another terminal 
run comman : cd backend  " to open backend folder in terminal"

run command: npm run dev " to start backend"
